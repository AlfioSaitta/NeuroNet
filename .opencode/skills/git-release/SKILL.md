## What I do
- Analizzo i commit recenti per estrarre le feature, fix e breaking changes
- Propongo un version bump (major/minor/patch) basato sui conventional commits
- Genero un changelog formattato
- Fornisco il comando `gh release create` da eseguire

## When to use me
Usa questa skill quando devi preparare una release taggata.
Chiedo chiarimenti se lo schema di versionamento non è chiaro.
